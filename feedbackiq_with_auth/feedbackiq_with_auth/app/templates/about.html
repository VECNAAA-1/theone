About FeedbackIQ

AI-powered customer feedback analysis system

🎯 Project Overview

FeedbackIQ is an Intelligent Customer Feedback Analysis System that leverages Natural Language Processing (NLP) and Machine Learning to automatically process large volumes of unstructured customer feedback and extract meaningful business insights.

⚙️ System Architecture

1
Data Collection
Accepts text input, CSV, TXT, or JSON files.
2
Text Preprocessing
Removes stop words, punctuation, URLs; lemmatizes tokens.
3
Sentiment Analysis
TextBlob-based polarity & subjectivity scoring.
4
Theme Extraction
TF-IDF keyword and n-gram phrase extraction.
5
Insight Generation
Rule-based recommendations, alerts, and summaries.
6
Visualization
Pie charts, bar charts, word clouds, histograms.

🛠️ Technology Stack

🐍 Python 3.11
🌶️ Flask 3.0
📚 NLTK
🤖 TextBlob
📊 scikit-learn
📉 Matplotlib
☁️ WordCloud
🎨 Vanilla JS + CSS
